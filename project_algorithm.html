Three-dimensional volume containing multiple two-dimensional information patterns

Hirotaka Nakayama, Atsushi Shiraki, Ryuji Hirayama, Nobuyuki Masuda, Tomoyoshi Shimobaba, and Tomoyoshi Ito,
Scientific Reports 3, 1931 (2013). [link]


Abstract

We propose an algorithm for recording multiple grayscale 2D patterns in a single 3D object. This algorithm offers high flexibility to the number of recorded patterns and those projection directions. In addition, each of the recorded patterns has directional characteristics; thus, can be revealed only when viewed from a certain direction. The algorithm is demonstrated with numerical simulations and actual prototype of a laser-engraved 3D cubic glass, which projects different three patterns in the orthogonal directions. Such 3D objects can be used as media art, digital signage, and information security.
